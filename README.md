## ✨ Features

- 🌌 **Cyber Network Background** — A fully custom Canvas2D particle engine with intelligent cluster networking and ultra-smooth drift animations
- 🖱️ **Interactive Cursor Network** — Particles near the cursor change color to red and form a dynamic web that follows your mouse in real-time
- 🎲 **3D Scroll Cube** — An animated rotating info cube in the bottom-left corner showing section previews
- 📊 **Animated Skill Rings** — Circular ring cards for programming and currently-learning skills
- 📅 **Timeline** — Clean vertical education and experience timeline
- 💼 **Project Cards** — Glassmorphism-style tech cards with hover parallax, glow effects, and live links
- 🏆 **Achievement Counters** — Animated stat counters for Projects, Certifications, Hackathons, and Internship
- 📬 **Working Contact Form** — Powered by Web3Forms; delivers submissions directly to email (no backend)
- 📱 **Fully Responsive** — Hamburger nav, fluid grids, and mobile-optimised layouts
- ♿ **Accessibility Ready** — Respects `prefers-reduced-motion` and uses semantic HTML5

## 🚀 Getting Started

No install steps, no npm, no dependencies. It's pure HTML.

### Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/TheCyberUchiha/Portfolio.git

# 2. Navigate into the project
cd Portfolio

# 3. Open in browser — just double-click index.html
# OR use a local dev server for best results:
npx serve .
# OR
python -m http.server 5500
```

Then open `http://localhost:5500` in your browser.

## 📬 Setting Up the Contact Form

The contact form uses **Web3Forms** — a free service that sends form data to your email with zero backend required.

**Steps:**
1. Visit **[web3forms.com](https://web3forms.com)**
2. Enter `the.cyber.uchiha.27@gmail.com` and click **Create Access Key**
3. Check your email for the access key
4. Open `index.html` and paste it here:

```html
<input type="hidden" name="access_key" value="PASTE_YOUR_KEY_HERE">
```

Every form submission will now be delivered to your inbox instantly, with the sender's name, email, subject, and message.

## 🎨 Background Engine — `three-bg.js`

A fully custom Canvas2D particle system — **no Three.js, no WebGL**.

| Feature | Details |
|---|---|
| Solo Particles | 120 independent dots drifting with organic motion |
| Clusters | 7 small groups of 4–10 particles connected by glowing lines |
| Cursor Attraction | Particles within 110px radius flow toward the mouse |
| Cursor Networking | In-range dots turn red and form a live network web |
| Performance | `requestAnimationFrame` loop, respects `prefers-reduced-motion` |
